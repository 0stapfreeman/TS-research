# TS-Research

Time series research: Fourier-based imputation and sparse recovery methods.

## Setup

```bash
# Install uv (if not installed)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create venv and install dependencies
uv sync

# Activate venv
source .venv/bin/activate

# Install with dev dependencies (jupyter, etc.)
uv sync --all-extras
```
